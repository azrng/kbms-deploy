                                                            # 介绍

                                                            图床工具，就是自动把本地图片转成连接的一款工具，网络上有很多图床工具，PicGo就是一款比较优秀的图床工具，可以支持微博，七牛云，腾讯云COS，又拍云，GitHub，阿里云OSS，SM.MSopen in new window，imgur 等8种常用图床

                                                            GitHub：https://github.com/PicGo/open in new window

                                                            # 安装

                                                            需要先安装nodejs

                                                            官网：https://nodejs.org/en/open in new window

                                                            从Github上下载PicGo工具，地址：https://github.com/Molunerfinn/PicGo/releasesopen in new window

                                                            下载合适系统的版本，下载安装

                                                            # Gitee图床

                                                            国内平台，访问速度快

                                                            登录gitee进入(没有账号的需要创建账号)

                                                            创建一个公开的仓库用户存储图片

                                                            进入个人设置配置私人令牌

                                                            配置好的令牌密钥需要保存好

                                                            打开PicGo搜索插件Gitee进行安装

                                                            然后我们可以再图床设置找到

                                                            配置参数

                                                            名称作用
                                                            URL填写gitee的官网网址
                                                            owner注册gitee时留的名字
                                                            repo仓库名
                                                            path存储的位置（不要是中文）
                                                            token刚才保存的私人令牌
                                                            message表述型文字（可以不填）

                                                            然后我们就可以上传一个测试下是否可以使用

                                                            上传后的图片显示在相册
